Graded 802.11n

Packet Delivery Modeling in Mobile 802.11n Networks

Download this project as a .zip file Download this project as a tar.gz file

Brief Description

Graded 802.11n is composed of packet delivery measurement on both network layer and device layer.

  1. The network layer conducts DSWA calculations to determine averaging intervals and sliding factor, and makes GradedT update to get rate selection indexes.
  2. On the device layer, it is driven by transmitting and receiving events that execute PDR computation and RSS averaging respectively. The rate indexes are also selected on device layer according to results of network layer when the PDR or RSS is lower than transmission threshold.

Files

  1. data: measured data including packet loss rate, packet failed rate, received signal strength, etc.
  2. DSWA: dynamic sliding window averaging
  3. gnuplot: demos of gnuplot of experimental results
  4. pktinfo: packet delivery measurement source files based on ath9k

How to

For DSWA

  1. cd DSWA
  2. sudo make
  3. sudo DSWA

For pktinfo

  1. make sure the device information in common.h is exactly what you are going to track
  2. cd pktinfo
  3. sudo make
  4. sudo pktinfo

For gnuplot

  1. make sure the experimental results and time.plt are in the same folder
  2. gnuplot
  3. load time.plt
